## 1. 安装扩展

1. 打开 Chrome，地址栏输入 `chrome://extensions/` 回车
2. 右上角打开「**开发者模式**」开关
3. 点击「**加载已解压的扩展程序**」
4. 选择项目的 `dist/` 目录（或解压后的 zip 目录）
5. 扩展图标出现在工具栏右侧，安装完成

> 如果之前加载过旧版，点击扩展卡片上的🔄**刷新**按钮更新。

---

## 2. 配置 API Key

**第一次使用必须先配置，否则翻译按钮无效。**

1. 点击工具栏的扩展图标，打开侧边栏
2. 点击右上角「⚙ 设置」Tab
3. 填写以下三个 Key（均来自 [1xm.ai](https://1xm.ai)，格式 `sk-...`）：

| 字段 | 用途 | 必填 |
|------|------|------|
| Vision API Key | OCR 识别图片文字 | 强烈建议 |
| Banana 2 Key | Nano Banana 2 模型生图 | 二选一必填 |
| Banana Pro Key | Nano Banana Pro 模型生图 | 二选一必填 |

4. 点「**保存设置**」

> Key 只存在你本地浏览器，不上传任何服务器。

## 3. 单图翻译

最快速的方式，适合临时翻译单张图片。

**操作步骤：**

1. 在任意网页上找到想翻译的图片
2. 在图片上**右键**
3. 选择「**🌐 翻译此图片 (Image Translator)**」
4. 侧边栏自动弹出，显示原图
5. 在侧边栏里选择：
   - **源语言**：自动检测 / 手动指定
   - **目标语言**：选你想翻译成的语言
   - **模型**：Banana 2（快）或 Pro（精细）
   - **品牌保护**：开启时保留 Logo / 型号，关闭时全量翻译
6. 点击「**翻译**」
7. 等待 **10–40 秒**，翻译结果出现在原图下方
8. 点击结果图可**全屏预览**，右上角点「↓ 下载」保存

## 4. 批量翻译

适合商品详情页、产品图册等需要翻译多张图的场景。

### 4.1 深度扫描

1. 打开目标网页（如京东商品页、天猫商详页等）
2. 点击扩展图标，打开侧边栏
3. 切换到「**批量**」Tab
4. 点击右上角「**↺ 深度扫描**」按钮
5. 等待 2–5 秒（扩展会自动滚动页面触发懒加载，识别所有商品图）
6. 图片网格出现，商品主图 / 商详图排在前面，噪音图（头像、广告、评论图）已过滤

### 4.2 选图

- 勾选网格里的图片（点击图片可切换选中状态，左上角显示 ✓）
- 右上角「**全选**」/ 「**取消**」快速操作
- 底部显示「已选 N 张」

### 4.3 开始批量翻译

1. 选完图后选择语言 / 模型
2. 点「**批量翻译**」
3. 自动跳转到「**结果**」Tab，可实时看每张进度
4. 翻译顺序执行（每张间隔 300ms，避免触发配额限制）
5. 遇到 429 配额超限会自动等待并重试

## 5. 📌 Pin 精选队列

当自动扫描的图片太多太杂，或你只想翻译特定几张图时，用 Pin 手动挑选。

### 5.1 开启 Pin 模式

切到「批量」Tab 后，Pin 浮层**自动开启**——不需要额外操作。

### 5.2 Pin 一张图片

1. 在网页上把鼠标移到目标图片上
2. 图片附近出现黑色圆角「**📌 Pin**」按钮（约 0.2 秒后显示）
3. 点击按钮，图片进入 Pin 队列
4. 侧边栏自动切到批量 Tab，Pin 的图出现在顶部「📌 已 Pin」区域（**金色边框**）
5. 同时页面上该图片会出现 📌 角标，下次悬停显示「✓ 已 Pin」，**避免重复添加**

### 5.3 管理 Pin 队列

| 操作 | 方法 |
|------|------|
| 删除单张 | 点击 Pin 区域中图片右下角的红色 **❌** 按钮 |
| 清空全部 | 批量 Tab 顶部「**清空 Pin**」按钮 |
| 查看角标 | 页面上已 Pin 的图右上角有 📌 标记 |

### 5.4 Pin + 扫描混合使用

Pin 的图片��在顶部（金色边框），扫描的图片排在下方——可以混合选择一起翻译。

## 6. 翻译结果查看与下载

### 查看结果

- 翻译完成后自动切到「**结果**」Tab
- 每条记录显示「原图 / 翻译后」对比
- 翻译中显示旋转动画，完成显示绿点，失败显示红点

### 放大预览

- 点击任意结果图（原图或翻译后图）→ 全屏 Lightbox 打开
- 按 **ESC** 或点击背景区域关闭

### 下载图片

- 方式一：Lightbox 打开后，右上角「**↓ 下载**」按钮
- 方式二：结果卡片右上角的「**↓ 下载**」按钮
- 下载格式为 PNG，文件名含翻译语言信息

### 清除记录

「结果」Tab 右上角「**清空全部**」按钮，清除本地缓存的翻译记录（不影响已下载的文件）。

## 7. 常见问题

**Q：点「深度扫描」没有图片出来？**

A：
1. 确认扩展已重新加载（`chrome://extensions/` 刷新）
2. 确认是通过点扩展图标打开侧边栏（让扩展绑定当前页面 Tab）
3. 某些特殊页面（chrome://、扩展商店页）不支持注入，换个普通网页试试

**Q：Pin 按钮不出现 / 只在部分图片上能 Pin？**

A：
1. 确认已切到「批量」Tab（离开批量 Tab 后 Pin 浮层会关闭）
2. 图片必须大于 80×80 px 才会触发
3. 有些图片被遮罩层挡住，扩展会用坐标穿透查找，但少数情况可能仍失效——这时候用深度扫描代替

**Q：翻译失败，提示「配额超限」？**

A：遇到 429 错误扩展会自动等待并重试一次。如果连续失败，等 1–2 分钟后再翻译，或减少单次批量数量（建议每批 ≤ 5 张）。

**Q：翻译结果品牌名被翻译掉了？**

A：在「设置」中开启「**品牌保护**」选项，Logo / 品牌名 / 型号会被识别后保留原文。

**Q：翻译很慢，要等多久？**

A：单张约 15–40 秒，Banana Pro 比 Banana 2 略慢但质量更高。批量翻译时间 = 张数 × 单张时间。

**Q：扫描到了不需要的图片（广告图、评论图等）？**

A：用 Pin 模式代替扫描，手动选你需要的图片；或者扫描后只勾选商品主图，忽略其他。
